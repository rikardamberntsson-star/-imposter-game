# Imposter – testbar webbapp

Det här är en enkel webbapp som kan testas direkt utan NAS och utan installation.

## Så testar du
1. Packa upp ZIP-filen.
2. Öppna `index.html` i en webbläsare.
3. Skriv in spelare, välj kategori och starta rundan.
4. Skicka mobilen mellan spelarna.

## Så fungerar spelet
- Alla spelare utom en får se samma ord.
- En spelare får se `???` och är impostor.
- Alla beskriver ordet med korta ledtrådar.
- Efter 1–2 varv röstar ni på vem som är impostor.
- Om impostern avslöjas får impostern försöka gissa ordet.

## Senare utveckling
Möjliga nästa steg:
- Poängräkning
- Fler imposters
- Timer
- Egna ordlistor
- Barnläge/familjeläge
- Delbar onlineversion
- Ljud och animationer
- PWA så den kan läggas på hemskärmen
